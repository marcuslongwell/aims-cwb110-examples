What's With the Dots?

In relative pathing, the dots are used to indicate which folder a file lives in. See the table below for some examples of the usage when traversing up directories with dots.

Path Browser's Understanding
./some-page.html Open a file that is in this same folder called "some-page.html"
../another-page.html Open a file that is in this up one directory called "another-page.html"
../../some-folder/my-page.html Go up two directories, then down into a directory called "some-folder" and open the file there called "another-page.html"
