Images

The img tag is used to display images in the browser.

Attributes

At minimum, you must have a src attribute on all images. However, there are a few other attributes to consider as well:

Attribute Purpose
src Specifies the hypertext reference for the image
alt Describes the image for the user in the event that the image cannot load
title Titles the image for accessibility & SEO
width Controls the width (in pixels) of the image
height Controls the height (in pixels) of the image

title Attribute

The title attribute should always be included to describe the image for accessibility and SEO purposes. Since blind people cannot see your image, for example, the title attribute helps describe to them what they would be seeing in a standard browser.

Furthermore, you can hover over the image to get more information about it even in a regular browser.

width/height Attributes

The width and height attributes can restrict the size of your image. Be careful, though, if you don't maintain the aspect ratio of the image, your image will be distorted!
