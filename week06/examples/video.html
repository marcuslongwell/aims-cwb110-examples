Video Size

The width and height attributes can control the size of the player.

Just like with <img> tags, you should normally only specify one or the other unless you know the aspect ratio of your video.

Video Autoplay, Muting, & Looping

The autoplay and loop attributes can enable you to play a video automatically whent the page loads, and have it loop back to the start when it ends, respectively. The muted attribute sets the video sound to muted by default. Note: many browsers block autoplay by default unless you enable it in your browser settings. Most browsers will not autoplay unless you also have the video muted by default.

Multiple Sources

It is alwasy recommended, if possible, to use multiple <source> tags within your <video> tags. That way, if the browser cannot play a type of file, you can always default it to one that it can play.

The type Attribute

The type attribute is an important attribute in your <source> tags. It specifies the type of file that the browser should be expecting, and helps the browser determine which of your sources to play.

File Format MIME-Type
MP4 video/mp4
WebM video/webm
Ogg video/ogg
