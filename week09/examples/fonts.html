CSS Fonts

If you remember back to our early weeks, you'll recall that there are two types of HTML tags: physical and logical. Physical tags change the appearance of text on a page. Logical change what the text means or how it should be interpreted.

In HTML5, logical tags are discouraged. Therefore, instead of using <font> tags to style our text, we should use CSS!
