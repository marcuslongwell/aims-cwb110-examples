Favicon

You've seen favicons all over the web. A favicon is an image used to represent the icon or logo of a web page or website.

Favicons are used in the bookmarks menu, in the search bar, in tabs, and everywhere else you can imagine needing a site icon for your website.

You can include a favicon in your page by using the <link> tag with a rel attribute of icon and an href attribute that defines a path to the image you want to use. Generally speaking, your icon should be either a png or an ico file.
