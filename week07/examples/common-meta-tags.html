Common Meta Tags

There are an unlimited number of meta tags, but the common ones are:

charset

Setting the charset attribute on a meta tag tells the browser what encoding you're including in your web page. It allows you to use utf-8 characters directly in your webpage, such as the Â¥ (yuan) symbol. This page is encoded in UTF-8. Note: some browsers will assume a charset intelligently, but we shouldn't rely on this..

description

A meta tag with a name of description and a content attribute will help browsers, search engines, and accessibility tools understand how your web page is described. Every web page should have a description.

keywords

A meta tag with a name of keywords and a content attribute will help browsers, search engines, and accessibility tools understand the main topics on a given page. You can think of it like a hashtag in social media!

author

A meta tag with a name of keywords and a content attribute will help browsers, search engines, and accessibility tools understand the who wrote a page. It can be a personal name, a company name, or some other entity that describes authorship for a page or information on a page.
