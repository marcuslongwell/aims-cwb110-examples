Audio Autoplay, Muting, & Looping

The autoplay and loop attributes can enable you to play a audio automatically whent the page loads, and have it loop back to the start when it ends, respectively. The muted attribute sets the audio sound to muted by default. Note: some browsers ignore the autoplay attribute, as it is considered bad practice.

Multiple Sources

It is alwasy recommended, if possible, to use multiple <source> tags within your <audio> tags. That way, if the browser cannot play a type of file, you can always default it to one that it can play.

The type Attribute

The type attribute is an important attribute in your <source> tags. It specifies the type of file that the browser should be expecting, and helps the browser determine which of your sources to play.

File Format MIME-Type
MP3 audio/mpeg
WAV audio/wav
OGG audio/ogg
