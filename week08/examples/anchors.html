Anchors

Anchor tags (<a>) are another example of a logical tag. Theyre logical because they tell the browser "hey, when a user clicks on the text in this tag, they should go to the hypertext reference". However, most browsers will still apply some default styles to an anchor tag. For example:

I'm blue, but when you click on me, I become purple

Nevertheless, we can set our own link color to match a theme if we wanted to. For example, I'm red! Click me!
