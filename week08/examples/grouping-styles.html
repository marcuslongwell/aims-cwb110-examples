Grouping Styles

If you remember a while back, we talked about how, often times, as a web developer, you want to put things in boxes.

At the time, it didn't make much sense. But luckily for us, now we can use those groups.

I'm in a black box with some padding.

I'm another paragraph in the same black box with some padding

Notice that all of this text is white, even though these paragraphs don't have any styles on them.

The reason they don't have to have styles on them is because they're grouped inside a box (a <div> tag), and the color style from that div tag cascades down to the paragraph tags.

But now, if I add another paragraph outside of the box, it no longer has the white text color.
